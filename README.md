# 🕐 Luxe Timepieces - Luxury Watch E-commerce Platform

A modern, responsive, and elegant e-commerce website showcasing premium luxury watches. Built with vanilla HTML, CSS, and JavaScript to ensure fast performance and smooth user experience.

## ✨ Features

### Core Functionality
- **Responsive Design** - Fully responsive layout that works seamlessly on desktop, tablet, and mobile devices
- **Product Catalog** - Browse through an extensive collection of luxury watches
- **Advanced Filtering** - Filter products by brand, price range, style, and more
- **Shopping Cart** - Add/remove products, update quantities, and calculate totals
- **Product Details View** - Detailed product information including specifications, images, and customer reviews
- **Newsletter Subscription** - Subscribe to receive exclusive deals and updates
- **Contact Form** - Get in touch with customer support

### User Experience
- Smooth animations and transitions
- Intuitive navigation
- Fast loading times (no external dependencies)
- Clean and modern UI design
- Mobile-first approach

## 🚀 Installation

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No server or build tools required

### Setup Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/athulraj13/Luxe-Timepieces.git
   cd Luxe-Timepieces
   ```

2. **Open in browser**
   - Simply double-click on `index.html`, or
   - Right-click and select "Open with" your preferred browser, or
   - Use a local server:
     ```bash
     # Using Python 3
     python -m http.server 8000
     
     # Using Python 2
     python -m SimpleHTTPServer 8000
     
     # Using Node.js (if http-server is installed)
     http-server
     ```
   - Then navigate to `http://localhost:8000` in your browser

3. **Start exploring!**
   - Browse the product catalog
   - Add items to your cart
   - Explore different features

## 📖 Usage

### For Users
1. **Browse Products** - Scroll through the product catalog or use the search/filter features
2. **View Details** - Click on any product to see detailed information
3. **Add to Cart** - Click "Add to Cart" button to add items
4. **Manage Cart** - Update quantities or remove items from your shopping cart
5. **Subscribe** - Enter your email to subscribe to the newsletter
6. **Contact** - Use the contact form to reach out with inquiries

### For Developers
- Modify `styles.css` to customize the visual appearance
- Update `script.js` to add new features or modify functionality
- Replace product images in the `images/` directory
- Update product data in `script.js` to add new watches
